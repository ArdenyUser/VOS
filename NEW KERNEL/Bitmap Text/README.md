## Usage :-<br/>

**Compile the source:**
```
	$ cd OS/NEW\ KERNEL/Bitmap\ Text
	$ make
```

**Open created ISO in QEMU:**
```
	$ qemu-system-x86_64 out/BitmapTest.iso
```
or
```
	$ qemu-system-i386 out/BitmapTest.iso
```
**Clean the code:**
```
	$ cd OS/NEW\ KERNEL/Bitmap\ Text
	$ make clean
```
