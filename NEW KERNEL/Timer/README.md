## Usage :-<br/>

**Compile the source:**
```
	$ cd OS/NEW\ KERNEL/Timer
	$ make
```

**Open created ISO in QEMU:**
```
	$ qemu-system-x86_64 out/Timer.iso
```
or
```
	$ qemu-system-i386 out/Timer.iso
```
**Clean the code:**
```
	$ cd OS/NEW\ KERNEL/Timer
	$ make clean
```
