## Usage :-<br/>

**Compile the source:**
```
	$ cd OS/NEW\ KERNEL/VESA\ VBE
	$ make
```

**Open created ISO in QEMU:**
```
	$ qemu-system-x86_64 out/VESA_VBE20.iso
```
or
```
	$ qemu-system-i386 out/VESA_VBE20.iso
```
**Clean the code:**
```
	$ cd OS/NEW\ KERNEL/VESA\ VBE
	$ make clean
```
