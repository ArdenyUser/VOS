
## Usage :-<br/>

**Compile the source:**
```
	$ cd OS/NEW\ KERNEL/Paging
	$ make
```

**Open created ISO in QEMU:**
```
	$ qemu-system-x86_64 -m 64 out/Paging.iso
```
or
```
	$ qemu-system-i386 out/Paging.iso
```
**Clean the code:**
```
	$ cd OS/NEW\ KERNEL/Paging
	$ make clean
```
